That java code-project covers [Data Structures & Algorithms in Java, 2/e by Robert W. Lafore](https://www.pearsoned.co.in/web/books/9788131718124_Data-Structures--Algorithms-in-Java_Robert-Lafore.aspx)

## Start

```sh
make
```

## Setup

```sh
make build
```

## Run

```sh
make run
```

## Run tests

```sh
make test
```

## Run checkstyle

```sh
make lint
```
